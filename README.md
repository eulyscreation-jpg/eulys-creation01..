
# Euly's Creation01 — GitHub Pages (Pink & Gold Luxury)

**What changed**
- Added **Pink & Gold luxury theme** (fonts, gradient hero, gold accents).
- Added **logo, phone number, social handles, and slogan** to the top of every page.
- Wired **product images** (filenames below) and kept variant dropdowns.
- Checkout and Contact use **mailto:** to send you an email (GitHub‑only approach; no server needed).

## 1) Upload these files to your repo (root)
Upload everything in this folder (the files you see here) straight to the repository **root** so that the root contains `index.html`, `styles.css`, etc.

## 2) Put your images into `/assets/` using these exact names
Rename your images on upload so they match the site:
```
logo.jpg                         ← your brand logo
bobo-balloon.jpg                 ← “bobo balloon.jpg”
bobo-balloon-basket.jpg          ← “bobo balloon basket.jpg”
funeral-shirts.jpg               ← “Funeral Shirts.jpg” (or Shirts)
regular-mugs.jpg                 ← “Regular Mugs.jpg” (product: Sublimation Mugs)
tumbler.jpg                      ← “Tumbler.jpg” (or “20oz Tumbler.jpg”)
shirts.jpg                       ← “Shirts.jpg”
glass-cups.jpg                   ← “Glass Cups.jpg” (used for Plastic Jars pics)
pillow.jpg                       ← “Pillow.jpg”
sublimation-picture-frames.jpg   ← “sublimation picture frames.jpg”
sublimation-puzzle.jpg           ← “sublimation puzzle.jpg”
tote-bags.jpg                    ← “tote bags.jpg”
memorial-buttons.jpg             ← “Memorial Buttons.jpg”
# Optional placeholders you can replace later if you have photos:
# dtf-logo-only.jpg, dtf-ready-prints.jpg, sublimation-keyrings.jpg
```
> The page expects **lowercase, hyphen** names. GitHub is case‑sensitive, so keep the names exactly as shown.

## 3) Turn on GitHub Pages
Repository **Settings → Pages** → **Deploy from a branch** → Branch: `main` • Folder: `/(root)` → **Save**.
Your site will appear at `https://<username>.github.io/<repo>/`.
